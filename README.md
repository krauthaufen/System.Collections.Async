# System.Collections.Async

An implementation of async enumerables with cancellation and exception handling.
Many operators are available out-of-the-box:

* Aggregate
* All
* Any
* AsEnumerable
* BufferedSource
* Canceled
* Concat
* Contains
* Count
* DefaultIfEmpty
* Distinct
* ElementAt
* ElementAtOrDefault
* Empty
* Except
* Faulted
* First
* FirstOrDefault
* ForEach
* FromValue
* FromValues
* Intersect
* Last
* LastOrDefault
* Latest
* LongCount
* Max
* Min
* Next
* Range
* Repeat
* Select
* SelectMany
* Single
* Skip
* SkipWhile
* Source
* Sum
* Take
* TakeWhile
* ToArray
* ToList
* Wait
* Where
